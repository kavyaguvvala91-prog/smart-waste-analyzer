# Objectives

- Detect waste items from user-uploaded images or camera captures.
- Provide AI-assisted guidance for reuse, recycling, reduction, and recovery.
- Allow users to report cleanup locations with geolocation and image evidence.
- Track eco points and user progress in a dashboard.
- Support municipality admins with report management and cleanup prioritization.
- Keep the experience public for browsing while protecting data-producing actions behind authentication.
