# Workflow

1. A user opens the public home page and reviews the features.
2. The user uploads an image or opens the camera to run waste detection.
3. The frontend sends the image to the detection pipeline and receives object results.
4. The analysis page summarizes detections and shows the pie chart.
5. The reuse and recycle page generates AI sustainability insights and DIY ideas.
6. The user can report cleanup locations with geolocation and earn eco points.
7. Admin users review reports, update status, and monitor dashboard metrics.
